Project Overview

Portofolio ini mencakup pengujian manual dan API untuk platform e-commerce "Second Hand". Tujuan utama dari pengujian ini adalah memastikan semua fungsi berjalan baik tanpa bug besar.

Test Execution:

Test scenario dan test case dibuat berdasarkan desain UI/UX untuk website dan aplikasi mobile. Pengujian dilakukan pada fungsionalitas seperti login, daftar produk, transaksi, dan pengelolaan pengguna.

Testing and Results:

Lebih dari 60 test case dieksekusi mencakup skenario positif dan negatif. Fokus pengujian pada:

  • Functional Testing: Memastikan fitur seperti pencarian produk, pemesanan, dan autentikasi berjalan lancar.
  • API Testing: Memastikan integrasi API menggunakan Postman dan Newman.
  • Performance Testing: Menguji performa API menggunakan JMeter untuk beban pengguna hingga 20 thread.
  • Cross-browser Testing: Memverifikasi kompatibilitas di berbagai browser.

Conclusion:

Pengujian menunjukkan sistem stabil dengan bug minor yang diperbaiki sebelum rilis. Fokus utama pengujian pada pendaftaran pengguna, transaksi produk, dan pengelolaan akun.
